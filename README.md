# QA Automation Project
 
This QA project will be integrated with a CI/CD pipeline (GitHub Actions) to enable automated test execution on every build/commit.
 
## Status
🚧 CI/CD integration in progress.
 
## Tech Stack
- Python
- Selenium WebDriver
- Pytest
- Page Object Model (POM)
## Planned CI/CD Integration
- Automated test runs triggered on push/pull request
- Cross-browser execution (Chrome, Firefox, Edge)
- Test reports and failure screenshots as build artifacts
